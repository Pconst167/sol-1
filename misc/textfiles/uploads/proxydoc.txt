Proxies by fris <fris@fris.net> Saturday October 18th, 2003

I thought I would make a document on the different uses of proxies, software to use, a better explaination.

Using a proxy is not always used by crackers, script kiddies and so called vandals of the "information super highway" so to speak. People now
adays like to use it for Anonymity like people who stand up for free speech, people who are investigating people.
Anonymity is just a tool of personal privacy online.

To better explain proxies. Read on. We will take your ip address say for example. 209.235.209.12. That resolves your point of presence 
on the internet. So basically your isp can monitor everything you do.

So, If you are using a proxy, you can put up a shield, to block that gateway between your isp and yourself. The proxy acts like a routing
communication device. They are mainly used for browsing the web.

There is Different software packages you can download, here I will go through several as follows.

Stealther:

Stealther is a program which you use together with your web browser to ensure 
your anonymity when you surf the Internet and download files. It allows you to hide your 
real identity from the websites you visit, block cookies and modify any information about 
your computer that is sent out by your web browser.

Stealther is a proxy server agent that lets you travel the 
Internet with maximum possible privacy. It contains a large autoupdated database of 
anonymous proxy servers and  allows you to easily logon to them and test their 
anonymity. 

This network of publicly accessible servers shield your current connection when you visit 
websites, download files, or use web-based e-mail accounts. You can also block cookies 
with a single click of your mouse, watch the statistics of the current connection and 
explore the information sent out by your browser.

You can scan each proxy server, check its response time, confirm its anonymity and IP 
isolation factors. Stealther  changes the proxy for each page automatically to 
ensure the maximum of security!

Stealther offers: 
modification of any HTTP variables, monitoring client requests, stop lists for clients and 
remote hosts, and much more.

This is a complete and expandable solution for both individual users and businesses 
concerned with their security and privacy of Web communications.


Stealther Website:

http://www.photono-software.de/Stealther/help/h_intro.php3

Proxy Switch:

Proxy Switch is designed for automatic switching of proxy servers used. It allows to distribute the load over a number of proxy servers 
as well as to remain anonymous on the Internet. 

Proxy Switch loads proxy server lists from standard text files and keeps successful/unsuccessful connection statistics for each of them. 
It can disable inoperable proxy servers automatically. Besides, general statistics for total volume of sent and received data, average and 
current transmission speed is kept. The friendly program interface allows monitoring the proxy servers used, disabling, enabling, and adding 
new ones in the real-time mode. 

The user can set proxy server switching criteria himself. These criteria include the number of connections to each proxy server, the time it 
has been used for, and the volume of data transmitted. 

Proxy Switch is capable of working with a very great number of simultaneous connections, system resource usage rate being very low. The program 
is operable under any Windows OS. 

Download Link:

mirror.mailutilities.com/psw.zip


YAPH: Yet another Proxy Hunter:

Dont run Windows? This is a proxy checker for Linux, and BSD. I know everyone doesn't approve of Bill. So Here is one for the other half.

YAPH is a proxy hunter for the Unix platform. It allows to find public access proxy servers on the Internet and to validate proxy lists. 
YAPH reveals SOCK4, SOCKS5, and HTTP (CONNECT method) proxies. HTTP proxies are tested for CONNECT method only, since only this method 
provides ability to tunnel TCP through HTTP proxy. YAPH utilizes the power of Nmap, a network mapper written by Fyodor. Nmap provides to YAPH 
the capability to find new undiscovered public proxy servers on the Internet. 

Download here: http://yaph.sourceforge.net/

Sending Anonymous MAIL:

Some people also wish to send emails anonymously. You could basically sign up for an account at www.hushmail.com which is a free encrypted
email service, Highly recommended among many IT professionals. Then if you have access to a web server with php then you can use the mail()
command, that is pretty much the best way to send email or email's. This way, it sends over localhost SMTP and so the IP address in the 
headers is your proxy address, so again your real ip address is filtered. So you are being anonymous.

EDONKEY, OVERNET, and EMULE:

This p2p file sharing program allows you to use SOCKS5 proxies to hide your ip as well.

ANONYMOUS FTP SOFTWARE:

FLASHFXP supports proxies, to connect between 2 ftp sites, your ip will remain anonymous.
