SimTel/msdos/virus/
ttrap132.zip    Trojan activity detector/protector

The Trojan Trapper is a utility which allows running a an executable
file in such a manner which prevents it from harming the system in
case it is a trojan. When a suspicious activity is detected the tested
program is stopped and an error message is displayed.  Please read
t-trap.doc before using.

Special requirements: None
