I. Usage
อออออออออ

   X-Tractor requires MS-DOS 3.3+ (or compatible).  and 64Kb Free memory
   X-Tractor was tested on most type of PC and seemed to work fine without
   any problems.

   TPCX  [Source <.EXE>] [Paramters] <ู

[Paramteres]

   -L  - List of known programs
   -B  - Make a Backup of the original file
   -I  - Identify program only
   -O  - Assume open on all queries
