                             BLUE BOX FREQUENCIES:
 
                      2600 HZ - USED TO GET ON/OFF TRUNK
 
                       TONE MATRIX TO USE AFTER 2600 HZ.
 
                           900 : 1100: 1300: 1500: 1700 :
 
                     700 :  1  :  2  :  4  :  7  :  11  :
 
                     900 :  +  :  3  :  5  :  8  :  12  :
 
                    1100 :  +  :  +  :  6  :  9  :  KP  :
 
                    1300 :  +  :  +  :  +  :  10 :  KP2 :
 
                    1500 :  +  :  +  :  +  :  +  :  ST  :
 
    USE KP (1700+1100) TO START A CALL AND KP2 THEN ST (1500+1700) TO STOP.
 
                          USE 2600 HZ TO DISCONNECT.
