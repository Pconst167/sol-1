ERICCSON POCKETPHONE

NOTES:  This is a dual NAM unit.
        The ESN prefix is 143 decimal, 8F hex.
        General Electric: 1-800-528-7711

PROGRAMING:

1.  Turn power on.

2.  Within ten seconds press and hold the FCN key and enter 9 2 3 8 8 5.

3.  The volume up key increments the step number.

4.  Use any numeric key to toggle on/off options and NAM 1/2 select.

5.  The END key completes programing and exits at any time.

PROGRAMING DATA:

STEP#    #OF DIGITS/RANGE    DISPLAY        DESCRIPTION

01       11 (UNCHANGEABLE)   SER NUMBER     DECIMAL ESN
02       4 DIGITS            LOCK CODE      LOCK CODE
03       ON/OFF              AUTO           AUTO SYSTEM REGISTRATION
04       ON/OFF              WHERE          AUTO ROAM REGISTRATION

05       1/2                 NAM x          TOGGLE NAM 1 AND 2

06       10 DIGITS           MIN            MIN (AREA CODE & TEL#)
07       00000 - 32767       SID            SYSTEM ID
08       ON/OFF              LOC            LOCAL USE MARK
09       ON/OFF              MRK            MIN MARK
10       333 OR 334          IPCH           INITIAL PAGING CHANNEL
11       2 DIGITS            ACCOLC         ACCESS OVERLOAD CLASS
12       A/B                 PREF           PREFERRED SYSTEM
13       2 DIGITS            GIM            GROUP ID (10 IN USA)

At this stage the phone will return to step 05, you can either:
Select the second NAM in which case steps 06 through 13 will repeat.
Scroll through and check your entries, or press END to complete programing.
