DIAMONTEL MESA 60, 60x, 80x

NOTES:  These are single NAM units.
        The ESN prefix is 134 decimal, 86 hex.
        See file MITSTEST.TXT for test mode access.
        DiamondTel: 708-298-9223

NAM programing:

1.  Press PWR key (FCN & 0 on model 60) for one second to turn on phone.

2.  Wait for the wake up cycle to end and the display to clear.

3.  Within ten seconds press and hold the CLR key while entering
    6 9 2 6 2 3 2 (reset code).

4.  Release CLR.

5.  The phone will display the first data entry.

6.  Press the SEND key to store information and increment the step number.

7.  Press CLR to correct any item.

8.  Press END to complete programing and exit.

PROGRAMING DATA:

STEP#    #OF DIGITS/RANGE    DISPLAY        DESCRIPTION

01       10 DIGITS           MUTE           MIN (AREA CODE AND TEL#)
02       4 DIGITS            SEC            SECURITY CODE
03       00000 - 32767       Sid            SYSTEM ID
04       0 OR 1              LU             LOCAL USE MARK
05       0 OR 1              E              MIN MARK
06       333 OR 334          IPCH           INITIAL PAGING CHANNEL
07       2 DIGITS            ACC            ACCESS OVERLOAD
08       0 OR 1              PrEF           PREFERRED SYSTEM
09       00 TO 15            GI             GROUP ID (10 FOR USA)
10       0 OR 1              EE             END TO END (DTMF) DIALING
11       0 OR 1              rI             ROAM INHIBIT
12       0 OR 1              Ab             PREFERRED SYSTEM
13       0 OR 1              AL             AUTO LOCK
14       0 OR 1              bO             HORN ALERT


NOTES:

To install lock code: With phone out of programing mode press FCN, STO,
NNNN LLL CLR (where NNNN is 4 digit security code and LLL is new lock code).

LOCK: F 2. UNLOCK: Enter three digit code then STO.

SYSTEM SELECT: F 1 displays current, then press one of the following:

"A" preferred phones:

0 = Pref A, 1 = Pref B, 2 = Home Only, 3 = A only, 4 = B only.

"B" preferred phones:

0 = Pref B, 1 = Pref A, 2 = Home Only, 3 = B only, 4 = A only.
